## 📋 Deskripsi

Website ini adalah portofolio pribadi mahasiswa Informatika yang menampilkan profil, keahlian, dan proyek-proyek yang telah dikerjakan. Dibuat sebagai tugas mata kuliah **Desain Antarmuka Pengguna (UI/UX)** Semester Genap 2025–2026.

## 👤 Target Pengguna

- Calon pemberi kerja / recruiter yang ingin melihat kemampuan
- Klien potensial untuk proyek freelance
- Rekan mahasiswa atau kolaborator
- Siapapun yang tertarik melihat karya dan pengalaman

## 🎯 Prinsip UI/UX yang Diterapkan

| Prinsip | Implementasi |
|---|---|
| **Konsistensi** | CSS Variables untuk warna, tipografi, dan spacing konsisten di seluruh halaman |
| **Hierarki Visual** | Ukuran font dan berat berbeda-beda untuk memandu mata pengguna |
| **Navigasi Jelas** | Sticky navbar, active state, smooth scroll, dan mobile hamburger menu |
| **Feedback** | Hover states, form validation real-time, loading indicator |
| **Keterbacaan** | Kontras tinggi, tipografi dipilih dengan hati-hati, line-height optimal |
| **Responsivitas** | Fluid grid, flexbox, CSS clamp — optimal di semua ukuran layar |
| **Aksesibilitas** | Semantic HTML, ARIA labels, focus states, alt text |
| **Performa** | CSS-only animations, lazy loading via IntersectionObserver |

## 🛠 Teknologi

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, CSS Grid, Animations
- **JavaScript (Vanilla)** — Interaksi dan animasi tanpa framework
- **Google Fonts** — Cormorant Garamond, DM Sans, Space Mono

## 🚀 Cara Menjalankan Lokal

```bash
# Clone repository
git clone https://github.com/Ozeeleon/portofolio_DAMP.git
cd portofolio_DAMP

# Buka dengan Live Server (VS Code Extension) atau:
npx serve .
```

## 🌐 Hosting

Website dihosting menggunakan **GitHub Pages**:
1. Push ke branch `main`
2. Aktifkan GitHub Pages di Settings → Pages → Source: `main`
3. Website otomatis tersedia di `https://username.github.io/portfolio`
